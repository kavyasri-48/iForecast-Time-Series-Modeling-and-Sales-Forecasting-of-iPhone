About This Project

iForecast is a student data analytics project focused on analyzing and forecasting Apple's iPhone sales performance in the Indian smartphone market. The project uses time-series modeling and machine learning techniques to analyze historical data and generate predictions for future iPhone sales and revenue growth up to 2036.


Through interactive Tableau dashboards and visual analytics, the platform provides insights into product trends, market share, revenue performance, and long-term growth patterns. These dashboards help users understand how iPhone sales have evolved over time and how future trends may shape the market.


Built using Flask and modern web technologies, this platform integrates data visualization, analytics, and predictive modeling to demonstrate how data science can be used to understand market behavior and forecast future business outcomes.


This project showcases practical skills in data analytics, data visualization, machine learning forecasting, and dashboard development.
